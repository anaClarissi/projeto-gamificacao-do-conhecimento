Front End

O Front-End é a área de desenvolvimento responsável pela criação da interface visual de sites e aplicações web. Os profissionais dessa área transformam layouts em páginas funcionais, cuidando da experiência do usuário, da interação e do design responsivo. Eles utilizam linguagens como HTML, CSS e JavaScript, frameworks e bibliotecas modernas para criar interfaces atraentes, eficientes e interativas.

Módulo 1 — Fundamentos de Front-End

  • O que faz um desenvolvedor Front-End
  • Estrutura de páginas web com HTML
  • Elementos semânticos
  • Links, listas e imagens
  • Formulários e inputs
  • Introdução a CSS: cores, fontes, tamanhos e margens

Módulo 2 — Estilizando com CSS

  • Aplicar cores, fontes, tamanhos e margens.
  • Trabalhar com bordas, sombras e espaçamentos.
  • Posicionar elementos com display, position, flexbox e grid.
  • Tornar o layout visualmente agradável e organizado.

Módulo 3 — Introdução ao JavaScript

  • Adicionar interatividade: cliques, teclado, movimento do mouse
  • Variáveis, operadores e funções simples.
  • Alterar conteúdo e estilo da página via JavaScript.
  • Eventos básicos (click, input, hover).

Módulo 4 — JavaScript Avançado

  • Trabalhar com listas (arrays) e objetos.
  • Métodos úteis (map, filter, forEach).
  • Validar formulários e entradas do usuário.
  • Armazenamento temporário com Local Storage.

Módulo 5 — Design e Boas Práticas

  • Princípios básicos de design → contraste, alinhamento, repetição e proximidade.
  • Boas práticas de layout → espaçamento correto, hierarquia visual e organização dos elementos.
  • Uso correto de cores → combinações harmoniosas e acessíveis.
  • Tipografia eficiente → escolha de fontes, tamanhos e legibilidade.
  • Componentes e padronização → manter consistência visual no projeto.
  • Boas práticas de CSS → organização, evitar repetições e uso de classes claras.
  • Acessibilidade → textos legíveis, boa navegação e elementos acessíveis para todos.

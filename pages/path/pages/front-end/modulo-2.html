Módulo 2 — Estilizando com CSS

Neste módulo, você aprenderá a transformar páginas HTML simples em sites bonitos e organizados, utilizando o poder do CSS. Vamos abordar passo a passo cada aspecto fundamental do CSS.

1. Introdução ao CSS

  • CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata) e é usado para definir o visual de uma página HTML.
  • Formas de aplicar CSS:
    • Inline: dentro da tag HTML (<p style="color: red;">Texto</p>).
    • Interno: dentro da tag <style> no <head>.
    • Externo: arquivo .css separado e vinculado via <link rel="stylesheet" href="style.css">.
  • Sintaxe básica:

    css

    seletor {
    propriedade: valor;
    }

    Exemplo:

    css

    p {
    color: blue;
    font-size: 16px;
    }

  • css

    /*Este é um comentário*/

2. Cores e fontes

  • É possível usar cores por nome, hexadecimal ou RGB:

    css

    color: red; /*nome*/
    color: #FF0000 /*hexadecimal*/
    color: rgb(255,0,0); /*RGB*/

  • Fontes: escolha fontes legíveis e coerentes com o design.

    css

    font-family: Arial, sans-serif;
    font-size: 16px;
    font-wight: bold;
    font-style: italic;
    text-transform: uppercase;

  • Aplicações: textos, títulos, links, botões e elementos de interface.

3. Margens, padding e bordas

  • Margin: controla o espaço fora de um elemento.

    css

    margin: 20px;

  • Padding: controla o espaço interno de um elemento.

    css

    padding: 10px;

  • Bordas: definem limites visíveis para os elementos.

    css

    border-width: 2px;
    border-style: solid;
    border-color: black;
    border-radius: 8px; /*bordas arredondadas*/

4. Layout e posicionamento

  • Tipos de exibição:
    • block: ocupa toda a largura disponível.
    • inline: ocupa apenas o espaço do conteúdo.
  • Posicionamento:
    • static: padrão.
    • relative: posição relativa ao lugar original.
    • absolute: posição em relação ao elemento pai.
    • fixed: posição fixa na tela.
  • Flexbox: organizar elementos horizontal ou verticalmente.

    css

    display: flex;
    justify-content: space-between;
    align-items: center;

  • Grid: criar layouts com linhas e colunas.

    css

    display: drid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;

5. Estilos avançados

  • Sombras:

    css

    box-shadow: 2px 2px 5px gray;
    text-shadow: 1px 1px 2px black;

  • Gradientes:

    css

    background: linear-gradient(to right, red, yellow);

  • Pseudo-classes: interatividade com o usuário.

    css

    a :hover {
    color: green;
    }
    button :active {
    background-color: blue;
    }

  • Pseudo-elementos: criar efeitos decorativos.

    css

    p::before {
    content: "• ";
    }

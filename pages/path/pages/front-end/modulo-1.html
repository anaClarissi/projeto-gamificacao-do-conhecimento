Front End

Módulo 1 — Fundamentos de Front-End

Neste módulo, você vai começar a explorar o mundo do desenvolvimento Front-End, que é a parte visual e interativa de um site. Você vai aprender o que faz um desenvolvedor Front-End e entender como as páginas da web são estruturadas.

1. O que faz um desenvolvedor Front-End

O desenvolvedor Front-End é responsável por criar e implementar a interface visual de sites e aplicações web. Ele transforma designs em páginas funcionais que os usuários podem interagir, garantindo que tudo seja responsivo, acessível e intuitivo. Além disso, precisa otimizar o desempenho e compatibilidade com diferentes navegadores e dispositivos.

2. Estrutura de páginas web com HTML

HTML (HyperText Markup Language) é a linguagem usada para estruturar conteúdos na web. Cada página web é composta por elementos HTML, como títulos, parágrafos, listas, imagens, links e formulários. É essencial entender a hierarquia de elementos, a organização do código e a correta utilização das tags.

3. Elementos semânticos

Elementos semânticos descrevem claramente o papel do conteúdo dentro da página. Por exemplo:

  • <header>: cabeçalho da página ou seção
  • <nav>: área de navegação
  • <main>: conteúdo principal
  • <section>: seções de conteúdo
  • <footer>: rodapé da página

O uso correto de elementos semânticos melhora a acessibilidade e o SEO do site.

4. Links, listas e imagens

  • Links: <a href="url">Texto do link</a> permite navegação entre páginas ou seções.
  • Listas: <ul> (não ordenada) ou <ol> (ordenada) organizam informações de forma hierárquica.
  • Imagens: <img src="caminho" alt="descrição"> insere imagens, sendo importante sempre usar o atributo alt para acessibilidade.

5. Formulários e inputs

Formulários coletam informações dos usuários e podem incluir:

  • Campos de texto (<input type="text")
  • Senhas (<input type="password")
  • Botões (<button ou <input type="submit")
  • Caixas de seleção (<select e <option)
  • Checkboxes (<input type="checkbox") e rádios (<input type="radio")

Formulários devem ser claros, acessíveis e fáceis de usar.

6. Introdução a CSS: cores, fontes, tamanhos e margens

CSS (Cascading Style Sheets) define a aparência visual da página. Alguns conceitos básicos:

  • Cores: color para texto, background-color para fundos
  • Fontes: font-family, font-size
  • Margens e espaçamentos: margin, padding
  • Bordas: border define a linha ao redor de elementos

CSS pode ser aplicado inline, em <style> no HTML ou em arquivos .css externos.

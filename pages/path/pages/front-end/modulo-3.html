Módulo 3 — Introdução ao JavaScript

O objetivo deste módulo é ensinar os alunos a dar os primeiros passos na linguagem que dá vida às páginas web: o JavaScript. Você aprenderá como o JS funciona, como criar scripts simples, como manipular elementos da página e como começar a programar de verdade.

1. O que é JavaScript? Para que ele serve?

JavaScript é a linguagem responsável por tornar sites dinâmicos e interativos. Enquanto o HTML estrutura a página e o CSS cria o visual, o JS controla o comportamento.

Exemplos de coisas feitas com JavaScript:

  • Mostrar ou esconder elementos (menus, pop-ups, alertas)
  • Validar formulários
  • Criar animações
  • Trabalhar com APIs
  • Criar jogos

Hoje JS é usado em sites, apps, jogos e até no backend, mas aqui trabalharemos apenas o JavaScript para front-end.

2. Inserindo JavaScript no HTML

Existem 3 formas principais de colocar JS no seu projeto:

  • JavaScript inline (dentro do HTML)

    html

    <button onclick="alert('Olá!')">Clique aqui</button>

    Usado apenas para coisas simples.

  • JavaScript interno (dentro da tag <script> no HTML)</strong>

    html

    <script>
    console.log("Página carregada!");
    </script>

  • JavaScript externo (arquivo separado .js)

    A forma mais profissional.

    HTML:

    html

    <script src="script.js"></script>

    script.js:

    javascript

    console.log("Arquivo JS conectado!");

3. Variáveis: let, const e var

Variáveis guardam valores.

As mais utilizadas no front-end atual são:

  • let — variável que pode mudar

    javascript

    let idade = 20;
    idade = 21;

  • const — variável que NÃO pode mudar

    javascript

    const nome = "Ana";

  • var — antiga, quase não se usa mais**

    Problemas com escopo → evitar.

4. Tipos de Dados

JS identifica automaticamente o tipo de cada variável:

Tipo Exemplos
String "Olá"
Number 10, 5.3
Boolean true, false
Array ["Ana", "Lucas"]
Object { nome: "Ana", idade: 15 }
Null valor propositalmente vazio
Undefined ainda não definido

Exemplo:

javascript

let nome = "Pedro"; // String
let idade = 17; // Number
let aprovado = true; // Boolean

5. Operadores

Matemáticos:

  • + soma
  • - subtração
  • * multiplicação
  • / divisão
  • % resto

Comparação:

  • === igual (valor e tipo)
  • !== diferente
  • > maior
  • < menor

Lógicos:

  • && E
  • || OU
  • ! NÃO

6. Condicionais (if, else if, else)

Usado para tomar decisões:

javascript

let idade = 16;

if (idade >= ) {
console.log("Maior de idade");
} else {
console.log("Menor de idade");
}

7. Funções — blocos de código que executam ações

Função simples:

javascript

function saudacao () {
console.log("Bem-vindo!");
}

saudacao();

Função com parâmetro:

javascript

function somar(a, b) {
return a + b;
}

console.log(somar(2, 3));

Função arrow:

javascript

const dobrar = x => x * 2;
console.log(dobrar(5));

8. Manipulação do DOM

O DOM é a "árvore" de elementos HTML.

Com JavaScript, podemos:

  • pegar elementos
  • mudar texto
  • mudar estilos
  • criar elementos
  • apagar elementos

Selecionar elementos:

javascript

document.getElementById("titulo");
document.querrySelector(".btn");

Alterar texto:

javascript

document.getElementById("titulo").textContent = "Novo Título!";

Alterar estilo:

javascript

document.querrySelector(".caixa").style.background = "blue";

Reagir a eventos:

javascript

document.(".btn").addEventListener("click", () => {
alert("Botão clicado!");
});

9. Eventos

Eventos são ações que acontecem na página:

  • click
  • mouseover
  • keydown
  • submit
  • change

Exemplo:

javascript

const btn = document.querrySelector("#enviar");

btn.addEventListener("click", () => {
console.log("Você clicou!");
});

10. Introdução à lógica aplicada no front-end

JavaScript é usado para:

  • validação de formulários
  • interações com o usuário
  • mostrar mensagens
  • fazer cálculos
  • criar animações simples
  • atualizar conteúdo dinamicamente

Exemplo de formulário simples:

javascript

const input = document.querrySelector("#nome");
const button = document.querrySelector("ok");

button.addEventListener("click", () => {
alert(`Olá, ${input.value}!`);
});

Back-End

Módulo 1 — Fundamentos de Back-End

Neste módulo, você vai aprender os conceitos fundamentais do desenvolvimento Back-End, entender como servidores, bancos de dados e APIs funcionam, e conhecer as linguagens mais utilizadas para criar a lógica de sistemas.

1. O que é Back-End

O Back-End é a parte de um sistema que processa a lógica, gerencia dados e garante que tudo funcione corretamente no servidor. Ele não é visível para o usuário, mas é essencial para que aplicativos e sites funcionem.

2. Arquitetura Cliente-Servidor

O modelo cliente-servidor separa responsabilidades: o cliente (front-end) exibe a interface, enquanto o servidor processa dados, armazena informações e responde às requisições. Essa comunicação é feita via protocolos como HTTP ou HTTPS.

3. Linguagens Comuns de Back-End

  • Node.js (JavaScript no servidor)
  • Python (Django, Flask)
  • PHP (Laravel, CodeIgniter)
  • Java (Spring Boot)
  • Ruby (Ruby on Rails)

4. Conceitos importantes

  • API: Permite que sistemas diferentes se comuniquem.
  • Request/Response: Requisição do cliente e resposta do servidor.
  • JSON: Formato padrão de troca de dados entre sistemas.

5. Ambiente de desenvolvimento

É importante configurar corretamente seu ambiente de Back-End. Instale a linguagem escolhida (Node.js, Python, etc.), configure sua IDE (VSCode recomendado) e aprenda a usar o terminal para executar scripts no servidor.

Módulo 4 — Redes de Computadores

Neste módulo você aprenderá os fundamentos das redes de computadores, incluindo tipos de redes, dispositivos utilizados, protocolos essenciais e conceitos de endereçamento IP.

1. O que é uma Rede de Computadores?

Uma rede de computadores é a interligação de dois ou mais dispositivos que compartilham dados, recursos e serviços através de um meio de comunicação.

2. Tipos de Redes

  • LAN: Rede local dentro de casas, escolas e empresas.
  • WAN: Redes de longa distância, como a internet.
  • MAN: Rede metropolitana (cobre uma cidade).

3. Principais Dispositivos de Rede

  • Roteador: distribui a internet e conecta redes diferentes.
  • Switch: conecta vários dispositivos dentro da mesma rede.
  • Modem: traduz o sinal da operadora para a internet.
  • Access Point: amplia o sinal Wi-Fi.

4. Protocolos Importantes

  • TCP/IP: base da comunicação na internet.
  • HTTP/HTTPS: protocolo de páginas web.
  • DNS: traduz nomes como "google.com" para IP.
  • DHCP: distribui IPs automaticamente.

5. Endereçamento IP

O endereço IP identifica cada dispositivo na rede. Pode ser IPv4 (ex.: 192.168.0.1) ou IPv6 (formato maior e mais moderno).

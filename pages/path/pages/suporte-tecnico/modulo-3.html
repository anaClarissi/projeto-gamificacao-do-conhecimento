Módulo 3 — Sistemas Operacionais

Neste módulo você aprenderá o que é um sistema operacional, como ele funciona, seus principais tipos, funções básicas e como realizar tarefas essenciais como gerenciamento de arquivos, usuários e permissões.

1. O que é um Sistema Operacional?

O sistema operacional (SO) é o software que controla o computador e permite que programas funcionem. Ele faz a ponte entre o usuário, o hardware e os aplicativos.

2. Exemplos de Sistemas Operacionais

  • Windows
  • Linux (Ubuntu, Mint, Fedora, etc.)
  • macOS
  • Sistemas móveis (Android, iOS)

3. Funções Principais do Sistema Operacional

  • Gerenciamento de memória
  • Gerenciamento de processos
  • Controle de dispositivos
  • Gerenciamento de arquivos

4. Sistema de Arquivos

Cada sistema operacional usa um tipo de sistema de arquivos para organizar e acessar dados, como NTFS (Windows), EXT4 (Linux) e APFS (macOS).

5. Permissões e Usuários

Sistemas operacionais permitem criar usuários, grupos e permissões, controlando quem pode acessar ou modificar cada arquivo.
